## SQL EDA Sales Analysis

An SQL Exploratory Data Analysis project focused on sales performance, customer behavior, and product insights.

### Objective
Analyze sales data using SQL to extract actionable business insights.

### Dataset
- Fact table: Sales
- Dimension tables: Customers, Products

### Tools Used
- SQL Server
- GitHub

### Key SQL Concepts
- SELECT, WHERE
- GROUP BY, HAVING
- JOINS
- Subqueries

### Outcome
Built hands-on experience in SQL-based exploratory data analysis and strengthened analytical thinking.
